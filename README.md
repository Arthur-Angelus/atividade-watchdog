# atividade wi-fi

Este projeto foi desenvolvido durante as aulas de IIoT do curso de Desenvolvimento de Sistemas no SENAI, com a funcionalidade de aprendermos sobre o sistema de WI-FI, roteador e esp32, o sistema vai mandar um codigo para a esp32 que fara com que a esp32 se conecte com um wi-fi/internet roteado atraves de nome e senha da internet, e os leds indicarão o estado de conexão se esta conectado ou desconectado

## como fazer
Para fazer o sistema seja fisicamente ou virtualmente você precisara de uma breadboard, um esp32, um ou mais resistores para cada led de pelo menos 220k, um ou mais leds da sua escolha e caso for fazer fisicamente precisara de um cabo para conectar a esp32 no computador e instalar o programa arduino para compilar o codigo e envia-lo para esp32.
